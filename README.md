## Description

discord-nestjs를 활용한 노트롤즈 서버 디스코드입니다. (개발중)

## Running the app

```bash
# development
$ npm run start

//서버에서 디스코드봇 역활을 위로 올려야합니다.

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
